# Color Boxes

A simple and visually engaging **Color Boxes** project built using **HTML5** and **CSS3**. This project demonstrates how different color formats and shadow effects can be used to create colorful UI components.

## Features

* Multiple colored boxes displayed in a grid layout
* Uses different CSS color formats:

  * Hexadecimal (`#ffff00`)
  * RGB (`rgb()`)
  * HSL (`hsl()`)
* Glowing box-shadow effects
* Rounded corners for a soft UI appearance
* Centered text labels inside each color box
* Responsive and clean layout

## Technologies Used

* HTML5
* CSS3

## Color Palette Included

| Color  | CSS Method                |
| ------ | ------------------------- |
| Yellow | Hex (`#ffff00`)           |
| Green  | RGB (`rgb(0,127,0)`)      |
| Red    | Named Color               |
| Blue   | HSL (`hsl(240,100%,50%)`) |
| Cyan   | Hex (`#00ffff`)           |

## Components

### Color Box

Each box contains:

* Colored background
* Glow shadow effect
* Inner sleeve label

### Sleeve

Acts as a label area inside each box and displays the color name.
